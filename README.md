# 📚 Books Scraper

A Python web scraper that crawls [books.toscrape.com](http://books.toscrape.com) and extracts structured data for all 1,000 books across 50 catalogue pages. Each book's detail page is also visited to pull the genre and UPC, giving you a rich, analysis-ready CSV dataset.

---

## Features

- **Full catalogue coverage** — iterates all 50 listing pages (1,000 books total)
- **Deep-detail scraping** — visits every individual book page to extract genre and UPC
- **Star-rating normalisation** — converts CSS class word ratings (`"Three"`) to integers (`3`)
- **Robust error handling** — gracefully skips pages or books that fail to load, logging the reason
- **Polite crawling** — 1-second delay between pages to avoid hammering the server
- **Clean CSV output** — saves structured data with `csv.DictWriter`, UTF-8 encoded

## Setup & Usage

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/books-scraper.git
cd books-scraper
```

### 2. Create and activate a virtual environment

```bash
python3 -m venv venv
source venv/bin/activate      # macOS / Linux
# venv\Scripts\activate       # Windows
```

### 3. Install dependencies

```bash
pip install requests beautifulsoup4
```

> No `requirements.txt` yet? You can generate one after installing:
> ```bash
> pip freeze > requirements.txt
> ```

### 4. Run the scraper

```bash
python main.py
```

The scraper will print progress as it works through the 50 pages. When finished, a `books_data.csv` file will be created (or overwritten) in the project directory.

## Output

### CSV columns

| Column | Description |
|---|---|
| `title` | Full book title |
| `price_gbp` | Price in GBP (numeric string, e.g. `51.77`) |
| `availability` | Stock status (e.g. `In stock`) |
| `rating` | Star rating as integer 1–5 |
| `genre` | Book genre extracted from breadcrumb navigation |
| `upc` | Universal Product Code from the product table |
| `url` | Direct link to the book's detail page |
