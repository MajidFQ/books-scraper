# 📚 Books Scraper

A Python web scraper that crawls [books.toscrape.com](http://books.toscrape.com) and extracts structured data for all 1,000 books across 50 catalogue pages. Each book's detail page is also visited to pull the genre and UPC, producing a clean, analysis-ready CSV dataset.

---

## Features

- **Full catalogue coverage** — iterates all 50 listing pages (1,000 books total)
- **Deep-detail scraping** — visits every individual book page to extract genre and UPC
- **Correct character encoding** — forces UTF-8 decoding on all responses, ensuring prices and titles are stored cleanly (no `Â£` artefacts)
- **Clean numeric prices** — strips the `£` symbol and stores prices as plain numbers (e.g. `53.74`), making the data immediately usable for sorting and analysis
- **Star-rating normalisation** — converts CSS class word ratings (`"Three"`) to integers (`3`)
- **Resume support** — tracks progress in `progress.txt`; if the run is interrupted, the next run picks up from the next unscraped page rather than starting over
- **Early-exit on completion** — if all 50 pages have already been scraped, the script detects this at startup and exits immediately without making any web requests
- **Crash-safe CSV writing** — appends each page's results to the CSV as it goes, so no data is lost if the script is interrupted mid-run
- **Robust error handling** — gracefully skips pages or books that fail to load, logging the reason
- **Polite crawling** — 1-second delay between pages to avoid hammering the server

## Setup & Usage

### 1. Clone the repository

```bash
git clone https://github.com/MajidFQ/books-scraper.git
cd books-scraper
```

### 2. Create and activate a virtual environment

```bash
python3 -m venv venv
source venv/bin/activate      # macOS / Linux
# venv\Scripts\activate       # Windows
```

### 3. Install dependencies

```bash
pip install requests beautifulsoup4
```

> To save your environment for others:
> ```bash
> pip freeze > requirements.txt
> ```

### 4. Run the scraper

```bash
python main.py
```

Progress is printed page by page. Results are written to `books_data.csv` as the scraper runs. If interrupted, simply run the same command again — it will resume from where it left off.

To start a completely fresh scrape, delete `progress.txt` and `books_data.csv` before running.

## Output

### CSV columns

| Column | Description |
|---|---|
| `title` | Full book title |
| `price_gbp` | Price in GBP as a plain number (e.g. `53.74`) |
| `availability` | Stock status (e.g. `In stock`) |
| `rating` | Star rating as integer 1–5 |
| `genre` | Book genre extracted from breadcrumb navigation |
| `upc` | Universal Product Code from the product detail table |
| `url` | Direct link to the book's detail page |

## Notes

- `books.toscrape.com` is a sandbox site built specifically for scraping practice.
- `progress.txt` is auto-generated at runtime. It is not required to exist before the first run.
- Prices are stored without the `£` symbol by design — the column name `price_gbp` captures the currency, and plain numeric values are easier to work with in analysis tools.
